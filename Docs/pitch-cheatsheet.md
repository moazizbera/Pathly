# Pathly Pitch Cheatsheet

Use this when you need to explain Pathly fast without collapsing into "task manager plus calendar" language.

## Main Point

Pathly is not a calendar app.

Pathly is a multi-role prioritization system.

Its value is that one person can manage different responsibilities through focused role planning, an `All Roles` view, visible reasoning, and role-aware next-step guidance.

## What Not To Say

Avoid leading with:

- we built a planner
- we built a task manager
- we built a smart calendar
- we help organize tasks on a dashboard

Those descriptions make Pathly sound replaceable.

## What To Say Instead

Lead with one of these:

- Pathly helps one person prioritize across multiple roles instead of forcing everything into one flat backlog.
- Pathly is built for users who are a Student, Employee, Teacher, or more than one of those at once.
- Pathly does not just store tasks. It helps the user decide what matters now and why.

## 10-Second Version

Pathly is a multi-role planning system that helps one person decide what to do next across different responsibilities.

## 30-Second Version

Most productivity tools are good at storing tasks but bad at helping people decide what matters first. That problem gets worse when one person is balancing multiple roles like Student, Employee, or Teacher. Pathly solves that by giving the user a focused role view, an `All Roles` overview, and a clear next move with visible reasoning.

## 60-Second Version

Most planners flatten everything into one generic list. But real people do not live in one role. A user might be a Student and an Employee at the same time, or a Teacher balancing instruction, grading, and planning work. Pathly treats that structure as the product itself. Tasks belong to a role context first, then the system builds an `All Roles` view on top. The dashboard recommends what to do next, explains why it matters, shows overlap between responsibilities, and keeps the same role logic across AI suggestions, task lanes, and the weekly calendar. So the value is not task storage. The value is multi-role prioritization.

## 2-Minute Version

Most productivity tools are built around storage. They let you capture tasks, assign dates, and maybe put items on a calendar. But they usually fail at the hardest part, which is deciding what matters first. That becomes even harder when one person is balancing multiple roles at once.

Pathly is designed for that exact problem. A user can be a Student, an Employee, a Teacher, or more than one of those at the same time. Instead of collapsing everything into one flat backlog, Pathly keeps one base profile, supports multiple role profiles, and attaches tasks to a real role context. From there, the product can do something a basic planner does not do well: it can switch between a focused role view and an `All Roles` view without losing structure.

That matters because the next best move changes depending on the role the user is in. In one view, the priority might be preparing a lesson. In another, it might be finishing a project review. In `All Roles`, the user can see planning pressure and overlap across responsibilities. Pathly also makes the reasoning visible through the next move card, risk radar, adaptive follow-up, role plans, and role-aware AI suggestions.

So the value is not that Pathly has a calendar. Many simple tools can do that. The value is that Pathly helps one person decide what deserves attention when life is split across different roles.

## Fast Judge Framing

Use this at the start of the demo:

"A normal planner stores work. Pathly decides what deserves attention when the same person is wearing different hats."

## Demo Order

Use this order in the spoken walkthrough:

1. explain the flat-backlog problem
2. show that one person can have multiple roles
3. show a focused role dashboard
4. switch to a different role and show a different next move
5. switch to `All Roles` and show overlap or pressure
6. open AI Suggest and show the role-aware grouping
7. show the calendar only as the execution layer

## Closing Line

Pathly helps overloaded people move from a messy list of responsibilities to one clear next step, across every role they live in.
